## 📋 Prerequisites
- Asegurarse tener Docker Desktop instalado. 

---

## 🚀 Usage
1. Build and start the containers using Docker Compose:
   ```bash
   docker-compose up -d
   ```
2. Access the web server at `http://localhost:8080`
